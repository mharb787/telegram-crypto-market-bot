# Telegram Crypto Market Bot

بوت تلغرام لتحليل أكبر 5 عملات كريبتو حسب القيمة السوقية مع استثناء العملات الثابتة. البوت يجمع بين التحليل الفني، سياق افتتاح وإغلاق الأسواق العالمية، فلتر BTC، وتعلم ذاتي محدود يضبط أوزان الاستراتيجية من نتائج التوصيات السابقة.

## ماذا يفعل؟

- يجلب أكبر العملات من CoinGecko ويستثني العملات الثابتة.
- يستخدم بيانات Binance لحساب مؤشرات 1h و 4h.
- يحسب EMA 20/50/200 و RSI و MACD و Bollinger Bands و ATR والدعم والمقاومة.
- يراعي نوافذ افتتاح وإغلاق آسيا وأوروبا وأمريكا.
- يعطي توصية مفصلة: شراء مشروط، مراقبة، انتظار، أو تجنب.
- يحدد وقف خسارة وأهداف مبنية على ATR والدعم والمقاومة.
- يسجل التوصيات ويتابع نتائجها.
- يعدل أوزان التحليل تدريجيًا من الأداء السابق بدل تغيير الكود عشوائيًا.

## التشغيل المحلي

انسخ ملف البيئة:

```bash
cp .env.example .env
```

ضع القيم:

```env
TELEGRAM_BOT_TOKEN=token_from_botfather
TELEGRAM_CHAT_ID=your_chat_id
ANALYSIS_INTERVAL_MINUTES=30
REPORT_INTERVAL_HOURS=6
MIN_CONFIDENCE_TO_ALERT=68
DRY_RUN=false
```

تشغيل فحص سريع:

```bash
npm run check
```

تشغيل تقرير واحد بدون إرسال تلغرام:

```bash
npm run once
```

تشغيل البوت:

```bash
npm start
```

## أوامر تلغرام

- `/start` يعرض أوامر البوت.
- `/report` يولد توصيات منفصلة، كل عملة برسالة مستقلة وزر بدء صفقة.
- `/status` يعرض أوزان الاستراتيجية وحالة ربط OKX.
- `/history` يعرض آخر التوصيات المسجلة.

## المراقبة التلقائية

البوت يفحص السوق تلقائيًا كل دقيقة بشكل صامت. لا يرسل رسائل في الوضع التلقائي إلا إذا كانت التوصية `شراء صريح`. أمر `/report` يبقى متاحًا للتحليل اليدوي ويرسل كل العملات بغض النظر عن قوة التوصية.

لمنع الإزعاج، لا يكرر البوت تنبيه `شراء صريح` لنفس العملة قبل مرور `STRONG_BUY_COOLDOWN_MINUTES`.

## تنفيذ الصفقات عبر OKX

عند وصول توصية منفصلة، يظهر زر `ابدأ صفقة حسب التوصية`. عند الضغط عليه يسأل البوت عن مبلغ الصفقة بالدولار، مثل `50`. بعدها يجلب السعر الحالي من OKX:

- إذا كان السعر قريبًا من سعر التوصية ضمن `MAX_PRICE_DRIFT_PERCENT`، يرسل أمر شراء Spot Market بقيمة المبلغ المدخل.
- إذا تغير السعر أكثر من الحد، لا يفتح الصفقة مباشرة ويسأل المستخدم هل يريد الاستمرار.
- إذا وافق المستخدم، يفتح الصفقة بالسعر الحالي ويستخدم وقف الخسارة والهدف الأول من التوصية.

يستخدم البوت OKX Spot Market order مع `tgtCcy=quote_ccy`، لذلك `50` تعني إنفاق 50 USDT. كما يرسل TP/SL مرفقة مع الأمر عندما تدعم OKX ذلك.

إعدادات OKX يجب أن تكون على الخادم فقط داخل `.env`:

```env
OKX_API_KEY=
OKX_API_SECRET=
OKX_API_PASSPHRASE=
OKX_SIMULATED_TRADING=false
MIN_TRADE_USDT=5
MAX_TRADE_USDT=500
MAX_PRICE_DRIFT_PERCENT=0.5
MAX_RECOMMENDATION_AGE_MINUTES=20
```

## النشر على VPS

1. ارفع المستودع على الخادم.
2. ثبّت Node.js 20 أو أحدث.
3. أنشئ ملف `.env` على الخادم فقط.
4. شغّل:

```bash
npm start
```

يفضل تشغيله عبر `systemd` أو `pm2` في الإنتاج.

## ملاحظات مهمة

- لا يتم تخزين توكن تلغرام أو بيانات VPS داخل GitHub.
- ملفات `data/*.json` تحتوي سجل التوصيات وضبط الاستراتيجية وهي مستثناة من Git.
- هذا تحليل آلي احتمالي وليس نصيحة مالية.
